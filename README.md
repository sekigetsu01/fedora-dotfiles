# Fedora Dotfiles

## Installation:

```
git clone https://github.com/sekigetsu01/fedora-dotfiles.git
cd ~/fedora-dotfiles
```

Alternatively, you could use [SFAI](https://github.com/sekigetsu01/SFAI).
This latter option is preferable.

## Note

I had a script to set my wallpaper with pywal, didn't work very well.
To change the wallpaper, change it in the `~/wallpapers` directory.
Then change it in the relevant sections in `~/.config/mango/config.conf`.
When you reboot, the wallpaper will change.
